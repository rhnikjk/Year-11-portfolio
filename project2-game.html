Implications

Aesthetics:
aesthetics is about making the game look good. the main thing for this project is making the text readable. we will do this by adding colour to make the more importaint things jump out. we will also add good line spacing.
functonality:
functionalty is about making the game functional, this means that we need to make sure it works. we will do this by testing every feature we add and making this in functions so it is easy to fix somthing if it goes wrong
usability:
usability is about making the game easy to use. we will do this by adding promts for the inputs the user can do;. we will also make sure that the user is clearly told what they can do at all times.
